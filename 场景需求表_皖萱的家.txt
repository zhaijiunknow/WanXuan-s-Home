二、详细场景需求

S01 寂静客厅初登场
对应剧本：1-1
地点：客厅
时间/天气：午后 / 晴
- 背景需求：大客厅、落地窗、午后自然光、沙发、茶几、墙钟、冰箱存在感、整体偏整洁宽敞略显冷清
- 角色：我（男主，可无立绘，仅旁白+文本）
- 道具：草莓蛋糕、蛋糕盒、塑料叉、手机
- 音乐/音效：轻柔钢琴，极淡；冰箱低鸣、空调送风、挂钟滴答、叉子碰纸盘轻响；阳台方向巨响、落地窗震动声
- 演出要点：重点表现安静、空、独居感；蛋糕特写建议做 CG 或食物插图；巨响处需要画面震动与节奏切换
- 可复用性：高

S02 阳台坠落
对应剧本：1-2
地点：阳台
时间：午后
- 背景需求：阳台外景、白天强光、地面有倒下的绿植、玻璃门与客厅连通感
- 角色：皖萱首次登场
- 角色造型需求：白色长裙、长发、半透明羽翼、头顶淡金色光环
- 表情需求：圣洁、无辜、尴尬、害羞、认真解释、饥饿发亮
- 道具：倒下的盆栽、蛋糕盘
- 音乐/音效：空灵/神秘短音色、肚子叫声、风声
- 演出要点：建议做首次登场高规格演出；吃蛋糕瞬间可做眼睛高光增强与微发光
- 可复用性：中高

S03 借宿协商
对应剧本：1-3
地点：客厅
时间：午后延续
- 背景需求：客厅同 S01，茶几上残留蛋糕盒痕迹，沙发对谈构图
- 角色：皖萱
- 表情：一本正经、期待、开心裂开、观察模式认真脸
- 道具：空蛋糕盒、羽毛通讯器、沙发、茶几
- 音乐/音效：轻松日常、略喜剧
- 演出要点：建立荒唐感与治愈感并存的同居起点
- 可复用性：高

S04 第一顿晚餐
对应剧本：1-4
地点：餐厅 / 厨房
时间：傍晚
- 背景需求：夕阳染色的餐桌、开放式厨房或厨房与餐厅切景、暖色晚霞光
- 角色：皖萱
- 道具/食物：三文鱼排、奶油蘑菇汤、蒜香意面、番茄生菜小盘、平底锅、锅铲、餐盘
- 音乐/音效：温暖木吉他；黄油融化、煎鱼滋啦声、餐具轻响
- 演出要点：需要做饭过程与餐桌成果两段视觉
- 可复用性：中高

S05 厨房共犯
对应剧本：2-1
地点：厨房
时间：白天
- 背景需求：明亮厨房、料理台、电饭锅、洗手池、食材摆台
- 角色：皖萱
- 表情/动作：认真学习、挫败、害羞脸红、距离拉近的慌乱
- 道具/食物：米饭、盐、海苔、金枪鱼罐头、蛋黄酱、玉子烧、味噌汤、腌黄瓜
- 音乐/音效：日常轻快 BGM、电饭锅保温嗡鸣、米饭团捏合声
- 演出要点：建议有手把手教做饭、擦面粉等亲密演出资源
- 可复用性：高

S06 外卖之夜
对应剧本：2-2
地点：客厅 / 玄关 / 餐桌
时间：傍晚 / 小雨
- 背景需求：雨天窗景版客厅、玄关开门构图、餐桌摆满外卖盒
- 道具/食物：手机外卖界面、纸袋外卖、寿司、炸鸡、奶茶、外卖塑料盒
- 音乐/音效：雨声、门铃、纸袋/塑料盒摩擦声
- 演出要点：需要外卖 UI 与开箱视觉
- 可复用性：中

S07 午后下午茶
对应剧本：2-3
地点：客厅
时间：午后 / 晴
- 背景需求：暖阳客厅、懒人沙发、茶几甜点盘、柔光与空气感
- 道具/食物：焦糖布丁、黄油曲奇、茶杯/咖啡杯、靠枕、薄毯
- 音乐/音效：慵懒轻爵士、杯碟轻响、风铃、时钟声
- 演出要点：强调“安静但不空”；可做午睡表情差分
- 可复用性：高

S08 深夜煮面
对应剧本：2-4
地点：厨房 / 餐桌
时间：深夜
- 背景需求：只开小灯的客厅与厨房、深夜室内暖光
- 道具/食物：挂面、鸡蛋、小青菜、叉烧、汤碗
- 音乐/音效：极轻钢琴/近乎无、水沸腾、切菜、面条入锅
- 演出要点：情感突破场，“因为是你陪我吃的”为高光台词
- 可复用性：高

S09 雨夜便利店
对应剧本：2-5
地点：便利店内 / 店外长椅
时间：夜 / 雨后
- 背景需求：便利店夜景、货架、加热柜、店外长椅与地面反光
- 道具/食物：牛奶、鸡蛋、吐司购物袋、关东煮、菠萝包、冰淇淋、塑料勺、纸杯
- 音乐/音效：夜晚街道音、便利店门铃、加热柜咕嘟声、雨后微风
- 演出要点：建议至少两张背景：店内与店外
- 可复用性：中

S10 日常蒙太奇
对应剧本：2-6
地点：客厅 / 厨房 / 阳台
时间：多时段
- 背景需求：冰箱门特写、厨房整理、阳台浇花、工作陪伴、早餐/零食/购物袋若干片段
- 关键道具：冰箱便签、零食、多肉植物、购物袋、手机点单界面
- 演出要点：适合做章节过场 montage，不必做单一完整场景
- 可复用性：高（偏过场资源）

S11 初夏夜宵
对应剧本：3-1
地点：客厅
时间：深夜 / 初夏
- 背景需求：夜色客厅、茶几临时餐桌、电视微亮、初夏夜风感
- 道具/食物：乌冬面、香菇、火腿、溏心蛋、葱花
- 音乐/音效：微妙电子氛围、深夜空调/风声、碗筷轻响
- 演出要点：转折前最柔软的一夜
- 可复用性：中高

S12 Yui 直播介入
对应剧本：3-2 ~ 3-3
地点：客厅 + 电视内直播间
时间：深夜
- 现实层背景：客厅夜景延续 S11
- 屏幕层背景：粉白配色、霓虹电子感、舞台感直播间、弹幕 UI、直播标题 UI
- 新角色：Yui
- Yui 风格：元气、电子偶像、神秘、可直视镜头
- 资源需求：直播界面 UI、弹幕滚动、电视屏幕滤镜、异常注视特效
- 音乐/音效：电子氛围、直播环境声、UI 切换音、异样静默点
- 演出要点：Yui 直视镜头时必须高规格演出
- 可复用性：中（重要度极高）

S13 直播后沉默
对应剧本：3-4
地点：客厅
时间：深夜
- 背景需求：电视熄灭后的昏暗客厅、小灯单点照明
- 角色：皖萱低头沉默差分
- 音乐/音效：无 BGM 或极低环境氛围
- 演出要点：情绪低压、真相前夜
- 可复用性：高

S14 阳台坦白
对应剧本：3-5
地点：阳台
时间：凌晨
- 背景需求：凌晨城市夜景、月光偏冷、风稍大、高处开放感
- 音乐/音效：空灵钢琴、风声、远处城市环境音
- 演出要点：核心设定揭露场，需要冷、静、真诚
- 可复用性：高

S15 召回通知
对应剧本：4-1
地点：客厅
时间：傍晚 / 雷雨将至
- 背景需求：天色压低的客厅、风吹窗帘、阳台风铃空响、雷雨前压迫感
- 道具：发光羽毛通讯器
- 音乐/音效：低缓弦乐、风声增强、通知提示音、冷静机械播报音
- 演出要点：客厅从日常切成危机场
- 可复用性：中高

S16 最后的普通晚餐
对应剧本：4-2
地点：厨房 / 餐厅
时间：傍晚到夜 / 下雨
- 背景需求：厨房暖灯、窗外下雨、餐桌小而温暖
- 道具/食物：饭团、味噌汤、玉子烧、米饭、海苔、金枪鱼蛋黄酱
- 音乐/音效：近乎无 BGM、雨声、开火、煮汤、鸡蛋滋声
- 演出要点：最后一顿平常饭，情绪需克制
- 可复用性：高

S17 选择前夜
对应剧本：4-3
地点：客厅 / 阳台门前
时间：夜 / 雨声持续
- 背景需求：雨夜客厅、阳台玻璃映出皖萱身影
- UI需求：玩家选项 UI（A 留下；B 回去）
- 音乐/音效：雨声主导、BGM 极弱
- 演出要点：正式选择点，可加入回忆闪回演出

三、角色资源需求汇总

1. 皖萱
- 基础形态：白裙、长发、羽翼、光环
- 人类留存形态（HE后）：无羽翼、无光环、家居服/普通服装版
- 常用表情：圣洁/平静、无辜、认真观察、开心发亮、害羞脸红、失落、困倦、哭泣、决意、温柔微笑
- 动作差分：坐沙发、抱靠枕、站厨房门口、捧碗、递食物、抓袖口、低头沉默、阳台回头

2. Yui
- 风格关键词：粉白电子偶像、元气、舞台感、神秘、可直视镜头
- 表情建议：营业笑、俏皮、注视、神秘、预言式微笑
- 资源建议：直播专用半身/Live2D、弹幕 UI 配套
