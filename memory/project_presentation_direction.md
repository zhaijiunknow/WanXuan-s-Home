项目当前更适合采用“3D场景叙事AVG + 2D角色演出”的表现路线：保留现有完整住宅3D场景（客厅、阳台、厨房、卧室/和室等）作为叙事舞台与空间连续性的基础，用皖萱的2D立绘/后续Live2D承载情绪表达与对话演出，而不是转回纯单背景VN。

**Why:** 用户已经具备较完整的3D住宅空间资源，以及已完成的人物立绘和表情差分；这种资源组合更适合低成本实现空间陪伴感、生活痕迹与剧情演出。

**How to apply:** 后续在给出产品形态、系统架构、美术统一方案、分镜和实现建议时，默认优先考虑“3D空间 + 2D角色”的整合路线；若使用AI生成2D背景，应优先作为关键CG或特写补充，而非替代现有3D主场景。
